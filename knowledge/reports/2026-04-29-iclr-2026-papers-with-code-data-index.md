# ICLR 2026 Papers with Code & Data Index

## Why it matters
Comprehensive index of ICLR 2026 accepted papers with linked code and data repositories, compiled as the Rio de Janeiro conference begins. Essential resource for practitioners who need to quickly find reproducible implementations of the latest accepted research. With Papers with Code discontinued in July 2025, this index fills a critical gap in the paper-to-code ecosystem.

## Lede (original)
The International Conference on Learning Representations (ICLR) 2026 event will be held in Rio de Janeiro, Brazil, starting at April 22nd. To facilitate rapid community engagement with the presented research, we have compiled an extensive index of accepted papers that have associated public code or data repositories.
