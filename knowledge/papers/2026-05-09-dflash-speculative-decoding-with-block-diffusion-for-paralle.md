# DFlash: Speculative Decoding with Block Diffusion for Parallel Token Drafting

## Why it matters
DFlash is a speculative decoding framework that uses a lightweight block diffusion model for parallel token drafting, achieving significant speedup over existing autoregressive methods while maintaining output quality. As inference costs dominate production LLM economics, techniques that accelerate generation without quality degradation are high-value. The approach is complementary to other inference optimizations like quantization and KV-cache compression.
