# VibeVoice: Long-Form Multi-Speaker Speech Synthesis

## Why it matters
VibeVoice synthesizes long-form multi-speaker speech using next-token diffusion and a highly efficient continuous speech tokenizer. Simon Willison tested the MLX conversion and achieved 8min 45sec processing time for an hour of audio on M5 Max MacBook Pro—practical for local transcription workflows. The model's combination of accuracy and efficiency makes it a strong option for developers building speech applications without cloud API dependencies.

## Abstract (original)
VibeVoice synthesizes long-form multi-speaker speech using next-token diffusion and a highly efficient continuous speech tokenizer, achieving superior performance and fidelity.
