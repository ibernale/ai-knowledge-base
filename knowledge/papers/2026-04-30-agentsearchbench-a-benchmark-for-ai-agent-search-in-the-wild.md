# AgentSearchBench: A Benchmark for AI Agent Search in the Wild

## Why it matters
As AI agent ecosystems grow, identifying suitable agents for tasks becomes a core challenge. AgentSearchBench is built from nearly 10,000 real-world agents across multiple providers and reveals a consistent gap between semantic similarity and actual agent performance. The finding that lightweight behavioral signals (execution-aware probing) substantially improve ranking quality has immediate implications for agent marketplace design and orchestration systems.

## Abstract (original)
The rapid growth of AI agent ecosystems is transforming how complex tasks are delegated and executed, creating a new challenge of identifying suitable agents for a given task. We introduce AgentSearchBench, a large-scale benchmark for agent search in the wild, built from nearly 10,000 real-world agents across multiple providers.
