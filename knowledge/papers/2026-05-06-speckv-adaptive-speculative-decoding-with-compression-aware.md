# SpecKV: Adaptive Speculative Decoding with Compression-Aware Gamma Selection

## Why it matters
SpecKV addresses a core inference efficiency problem: how to adaptively select the gamma parameter in speculative decoding while being aware of KV-cache compression. Speculative decoding is becoming standard for LLM serving at scale, and papers that improve the accuracy-latency tradeoff directly impact production costs. Code and data are available, making this immediately useful for inference engineers.
