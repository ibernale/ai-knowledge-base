# Agent Context Files: A Large-Scale Empirical Study

## Why it matters
First large-scale empirical study of 2,303 agent context files ('READMEs for agents') from 1,925 repositories. Finds these files evolve like configuration code through frequent small additions, not static documentation. Developers prioritize functional context (build/run commands 62.3%, implementation details 69.9%, architecture 67.7%) while neglecting non-functional requirements (security 14.5%, performance 14.5%). Essential reading for anyone building or managing agentic coding systems.

## Abstract (original)
Agentic coding tools receive goals written in natural language as input, break them down into specific tasks, and write or execute the actual code with minimal human intervention. Central to this process are agent context files ('READMEs for agents') that provide persistent, project-level instructions. In this paper, we conduct the first large-scale empirical study of 2,303 agent context files from 1,925 repositories to characterize their structure, maintenance, and content.
