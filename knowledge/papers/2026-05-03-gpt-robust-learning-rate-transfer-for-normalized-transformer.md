# νGPT: Robust Learning Rate Transfer for Normalized Transformers

## Why it matters
Introduces νGPT parametrization for Normalized Transformers (nGPT) enabling robust learning rate transfer across model width, depth, and token horizon. Addresses a key practical challenge in scaling: hyperparameter tuning costs. The modification to nGPT's scaling rules, informed by empirical alignment exponent measurements, significantly reduces computational burden for large-scale model training.

## Abstract (original)
A new parametrization named 'νGPT' for Normalized Transformers (nGPT) introduces robust learning rate transfer capabilities across model width, depth, and token horizon. This modification to nGPT's scaling rules significantly reduces the computational burden of hyperparameter tuning for large-scale models.
