# Synthetic Computers at Scale for Long-Horizon Productivity Simulation

## Why it matters
This Microsoft Research paper tackles a critical gap in agent evaluation: how to measure long-horizon productivity tasks that span hours or days. The 'synthetic computer' approach creates reproducible, scalable simulation environments for testing agent performance on complex multi-step workflows. This matters because current benchmarks mostly measure short tasks; real-world agent deployments require sustained autonomous execution. The work provides infrastructure for evaluating agents on realistic productivity scenarios, which is essential as coding and computer-use agents move toward longer autonomous runs.

## Abstract (original)
Preview version; work in progress. This paper introduces synthetic computers at scale for simulating long-horizon productivity tasks, enabling rigorous evaluation of AI agents on multi-step workflows that better reflect real-world deployment scenarios.
