# Workspace-Bench 1.0: Benchmarking AI Agents on Workspace Tasks with Large-Scale File Dependencies

## Why it matters
Introduces Workspace-Bench 1.0, a benchmark for evaluating AI agents on realistic workspace tasks involving large-scale file dependencies. Unlike existing benchmarks focused on single-file or simple contexts, this tests agents' ability to navigate complex organizational file structures, cross-file dependencies, and multi-step workflows that mirror real enterprise environments. Essential for practitioners evaluating agentic systems for deployment in document-heavy organizations.

## Abstract (original)
We present Workspace-Bench 1.0, a benchmark for evaluating AI agents on workspace tasks with large-scale file dependencies. The benchmark tests agents' ability to navigate complex organizational file structures, cross-file dependencies, and multi-step workflows that mirror real enterprise environments.
