# AutomationBench: Evaluating AI Agents on Cross-Application Workflow Orchestration

## Why it matters
AutomationBench addresses a critical gap in agent evaluation: cross-application coordination, autonomous API discovery, and policy adherence combined. Drawing on real workflow patterns from Zapier's platform across Sales, Marketing, Operations, Support, Finance, and HR domains, it requires agents to discover endpoints, follow layered business rules, and write correct data to multiple systems. Even frontier models score below 10%, establishing a realistic baseline for what businesses actually need from agentic automation.

## Abstract (original)
Existing AI benchmarks for software automation rarely combine cross-application coordination, autonomous API discovery, and policy adherence. Real business workflows demand all three. To address this gap, we introduce AutomationBench, a benchmark for evaluating AI agents on cross-application workflow orchestration via REST APIs.
