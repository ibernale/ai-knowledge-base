# Evaluating AGENTS.md: Are Repository-Level Context Files Helpful for Coding Agents?

## Why it matters
Rigorous evaluation finding that context files (AGENTS.md) actually reduce task success rates by ~3% while increasing inference costs by 20%+. Counterintuitive result challenges the widespread industry practice of using context files for coding agents. The finding that unnecessary requirements make tasks harder suggests practitioners should specify only minimal requirements—important guidance for teams deploying coding agents.

## Abstract (original)
A widespread practice in software development is to tailor coding agents to repositories using context files, such as AGENTS.md, by either manually or automatically generating them. Although this practice is strongly encouraged by agent developers, there is currently no rigorous investigation into whether such context files are actually effective for real-world tasks. Across multiple coding agents and LLMs, we find that context files tend to reduce task success rates compared to providing no repository context.
