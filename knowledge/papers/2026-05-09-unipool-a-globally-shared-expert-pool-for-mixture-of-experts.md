# UniPool: A Globally Shared Expert Pool for Mixture-of-Experts

## Why it matters
UniPool proposes a globally shared expert pool architecture for Mixture-of-Experts models, addressing the parameter inefficiency of traditional per-layer expert allocation. By sharing experts across layers, the approach achieves better parameter utilization while maintaining routing flexibility. This is relevant for practitioners deploying large MoE models where memory constraints are binding—the shared pool design could enable larger effective model capacity within the same memory footprint.

## Abstract (original)
A globally shared expert pool for Mixture-of-Experts that improves parameter efficiency by sharing experts across layers while maintaining routing flexibility.
