# Nemotron 3 Nano Omni: Efficient Omni-Modal Model for Edge Deployment

## Why it matters
NVIDIA introduces Nemotron 3 Nano Omni, an omni-modal model capable of processing text, images, video, and native audio. Achieves consistent accuracy improvements across modalities with up to 7.5x higher inference throughput in NVFP4 on B200 GPUs compared to BF16 for single-image reasoning. Less than 1% median accuracy drop when quantized to FP8 and NVFP4. Demonstrates the path to practical multimodal deployment on NVIDIA's next-gen inference hardware.

## Abstract (original)
NVIDIA introduces Nemotron 3 Nano Omni, an omni-modal model capable of processing text, images, video, and native audio, achieving consistent accuracy improvements across modalities and up to 7.5x higher inference throughput in NVFP4 on B200 GPUs compared to BF16 for single-image reasoning. The model demonstrates robust performance with less than 1% median accuracy drop when quantized to FP8 and NVFP4.
