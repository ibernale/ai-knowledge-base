# ARIS: Open-Source Research Harness Using Cross-Model Adversarial Collaboration

## Why it matters
ARIS is an open-source research harness that uses cross-model adversarial collaboration to ensure reliable long-term research outcomes. The system coordinates execution, orchestration, and assurance layers across multiple models, using adversarial dynamics to catch errors that single-model systems miss. This architecture is relevant for autonomous research agents where reliability over long horizons is critical—the adversarial design provides a form of built-in verification.
