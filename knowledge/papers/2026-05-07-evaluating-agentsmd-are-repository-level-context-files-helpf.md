# Evaluating AGENTS.md: Are Repository-Level Context Files Helpful for Coding Agents?

## Why it matters
Accepted as Oral & Runner-up Best Paper at MemAgents@ICLR 2026, this ETH Zurich study finds that AGENTS.md context files result in no improvement in task success rates while increasing inference cost by over 20%. LLM-generated files actually reduce success rates by ~3%. Developer-written files offer marginal ~4% gains but with 19% cost increases. This challenges the widespread industry recommendation to use context files and suggests practitioners should measure rather than assume benefit.

## Abstract (original)
A widespread practice in software development is to tailor coding agents to repositories using context files, such as AGENTS.md, by either manually or automatically generating them. Although this practice is strongly encouraged by agent developers, there is currently no rigorous investigation into whether such context files are actually effective for real-world tasks. Across multiple coding agents and LLMs, we find that context files result in no improvement in task success rates, while also increasing inference cost by over 20%.
