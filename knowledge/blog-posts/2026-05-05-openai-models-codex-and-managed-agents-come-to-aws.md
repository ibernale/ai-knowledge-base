# OpenAI models, Codex, and Managed Agents come to AWS

## Why it matters
AWS announces OpenAI models (GPT-5.5, GPT-5.4) available on Amazon Bedrock, plus Codex and OpenAI-powered Managed Agents. This marks a significant shift in the OpenAI-Microsoft exclusive relationship—developers can now access OpenAI models through AWS with unified security, governance, and cost controls. Codex usage counts toward AWS cloud commitments. For enterprises locked into AWS, this removes a major barrier to adopting OpenAI's agentic coding tools.

## Lede (original)
OpenAI models on Amazon Bedrock (Limited preview) brings the latest OpenAI models, including GPT-5.5 and GPT-5.4, to the Bedrock APIs you already use, with unified security, governance, and cost controls.
