# Anthropic Claude Code rate limits doubled, SpaceX Colossus partnership announced

## Why it matters
Anthropic announces a partnership with SpaceX to use 'all the capacity of their Colossus One data center'—over 300 megawatts and 220,000+ NVIDIA GPUs within the month. This addresses Anthropic's 80-fold Q1 revenue growth that CEO Dario Amodei called 'just crazy' and 'too hard to handle.' Effective immediately: doubled Claude Code five-hour rate limits for Pro/Max/Team/Enterprise, removed peak-hours reduction for Pro/Max, and raised Opus API limits. Anthropic also expressed interest in developing 'multiple gigawatts of orbital AI compute capacity' with SpaceX.

## Lede (original)
We've agreed to a partnership with SpaceX that will substantially increase our compute capacity. This, along with our other recent compute deals, means that we've been able to increase our usage limits for Claude Code and the Claude API.
