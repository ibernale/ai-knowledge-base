# GPT-5.5 Now Available on Databricks via Unity AI Gateway

## Why it matters
GPT-5.5 is now natively available on Databricks across AWS, Azure, and GCP, governed through Unity AI Gateway. This extends OpenAI's multi-cloud strategy beyond the AWS Bedrock announcement. Organizations can use GPT-5.5 for coding workflows in Codex, build agents grounded in enterprise data via Unity Catalog, and embed intelligence into document pipelines. Genie (Databricks' NL analytics) now uses multiple frontier LLMs including GPT-5.5. This shows the rapid fragmentation of OpenAI distribution channels post-Microsoft exclusivity.

## Lede (original)
GPT-5.5 is available now on Databricks across AWS, Azure, and GCP. Use GPT-5.5 to power coding workflows in OpenAI's Codex or any coding agent, build agents grounded in enterprise data, add deeper intelligence to document pipelines, and bring data insights into every employee's daily work.
