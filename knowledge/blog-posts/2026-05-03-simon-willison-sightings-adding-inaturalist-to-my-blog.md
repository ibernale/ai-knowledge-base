# Simon Willison: Sightings - Adding iNaturalist to My Blog

## Why it matters
Simon Willison demonstrates practical agentic coding by building a feature entirely on his phone using Claude Code for web while camping. He created an iNaturalist integration for his blog using a Git scraping repository pattern, showing how vibe-coding accelerates app development to the point where 'shipping a tool is more like posting on a blog.' The implementation chain—Python CLI, Git scraping repo, JavaScript frontend, blog syndication—illustrates the new workflow patterns emerging for AI-assisted personal tool development.

## Lede (original)
I have a new camera (a Canon R6 Mark II) so I'm taking a lot more photos of birds. I share my best wildlife photos on iNaturalist, and based on yesterday's successful prototype I decided to add those to my blog. I built this feature on my phone using Claude Code for web, as an extension of my beats system for syndicating external content.
