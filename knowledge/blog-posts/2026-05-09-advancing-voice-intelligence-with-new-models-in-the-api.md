# Advancing voice intelligence with new models in the API

## Why it matters
OpenAI introduces three new real-time voice models: GPT-Realtime-2 (first voice model with GPT-5-class reasoning), GPT-Realtime-Translate (live translation across 70+ input languages to 13 output languages), and GPT-Realtime-Whisper (streaming speech-to-text). GPT-Realtime-2 scores 15.2% higher on Big Bench Audio and 13.8% higher on Audio MultiChallenge vs GPT-Realtime-1.5. This shifts real-time audio from call-and-response toward voice interfaces that can reason, translate, transcribe and take action as conversations unfold—enabling a new class of voice-driven agents.

## Lede (original)
We're introducing three audio models in the API that unlock a new class of voice apps for developers. With these models, developers can build voice experiences that feel more natural, respond more intelligently, and take action in real time.
