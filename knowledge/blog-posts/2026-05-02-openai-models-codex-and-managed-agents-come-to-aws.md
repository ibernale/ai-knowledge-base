# OpenAI models, Codex, and Managed Agents come to AWS

## Why it matters
OpenAI has launched GPT-5.5, Codex, and Managed Agents on Amazon Bedrock, marking the end of Microsoft's exclusivity that expired April 27, 2026. This is backed by a $50B Amazon investment and represents the first multi-cloud distribution of OpenAI's frontier models. Enterprises can now use OpenAI models through existing AWS credentials, IAM policies, and billing. Codex on Bedrock allows organizations to apply usage toward AWS cloud commitments. Bedrock Managed Agents combines OpenAI's harness with AWS infrastructure for production-ready agent deployment. This fundamentally changes the competitive landscape for enterprise AI procurement.

## Lede (original)
For many companies, using AI at scale requires bringing the best models to the systems their teams already use. That's why we're launching OpenAI models, including our best frontier model GPT‑5.5, on Amazon Bedrock. Customers can now build with OpenAI models in AWS, alongside the services, security controls, identity systems, and procurement processes they already rely on.
