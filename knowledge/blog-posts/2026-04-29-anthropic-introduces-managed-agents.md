# Anthropic Introduces Managed Agents

## Why it matters
Anthropic launches Managed Agents as a hosted Claude Platform service for long-horizon agent work. Provides stable interfaces for sessions, harnesses, and sandboxes with emphasis on durable state, safer tool access, and faster startup for long-running tasks. Reflects Anthropic's strategy of moving from models to infrastructure for production agent deployment. Key insight: harness design assumptions need constant re-evaluation as model capabilities improve.
