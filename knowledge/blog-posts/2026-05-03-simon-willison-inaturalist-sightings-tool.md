# Simon Willison: iNaturalist Sightings Tool

## Why it matters
Demonstrates Claude Code for web as a viable mobile development environment. Willison built a complete observation-grouping system (inaturalist-clumper) from his phone while camping, combining Python CLI, Git scraping, and JavaScript visualization. The workflow shows how 'clumping' observations by proximity (2 hours, 5km) creates meaningful data structures that can be consumed via CORS. This is a practical case study in building lightweight personal tools with AI assistance.

## Lede (original)
I wanted to see my iNaturalist observations - across two separate accounts - grouped by when they occurred. I'm camping this weekend so I built this entirely on my phone using Claude Code for web. I started by building an inaturalist-clumper Python CLI for fetching and 'clumping' observations.
